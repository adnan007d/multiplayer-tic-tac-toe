# Shit

This is just a shit game for project made with socketio

Make sure Node.js is installed

Server

- Cd into the server directory
- Install dependencies with yarn

```zsh
yarn
```

- Run the development server

```zsh
yarn dev
```

Client

- cd into the client directory
- Install dependencies with yarn

```zsh
yarn
```

- Run the development server

```zsh
yarn dev
```
